# Controlled Substance Map Backend

Minimal Express backend for resolving substance names, querying legal status, and caching results in Supabase.

## Run

```bash
npm install
npm start
```

## Environment

Create a `.env` file with these keys:

- `HUGGING_FACE_API_KEY`
- `OPENAI_KEY`
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `PORT` (optional)
